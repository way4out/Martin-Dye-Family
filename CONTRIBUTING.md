# Contributing to the Martin Dye Family

Thank you for your interest in contributing to the **Martin Dye Family** project!

This is an open-science initiative dedicated to advancing rigidified polymethine cyanine dyes for NIR to SWIR applications. We welcome contributions from researchers, chemists, developers, and enthusiasts.

## Ways to Contribute

### 1. Reporting Issues or Suggestions
- Bug reports in synthesis protocols
- Ideas for new applications or improvements
- Questions about specific dyes (Tucyanin-F(S), AetherVoid-5S, etc.)

Please open a new **Issue** with as much detail as possible.

### 2. Submitting Code / Protocol Improvements
1. Fork the repository
2. Create a feature branch (`git checkout -b improvement/your-feature`)
3. Make your changes
4. Test your modifications thoroughly
5. Commit with clear, descriptive messages
6. Open a **Pull Request** with a detailed explanation

### 3. Areas We Especially Welcome
- Improved or safer synthesis routes
- Higher yields or lower-cost methods
- New characterization data (NMR, UV-Vis-NIR, photostability, toxicity)
- Real-world application testing and feedback
- Documentation enhancements
- New shelf-stable variants or derivatives
- Bug fixes and optimizations

### Code of Conduct
- Be respectful, constructive, and inclusive
- Maintain scientific integrity and safety standards
- Properly attribute all contributions
- Follow local laws and safety regulations for chemical work

### Attribution
All contributors will be acknowledged in the README and release notes. Significant contributions may result in co-authorship on future publications.
