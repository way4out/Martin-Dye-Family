# Martin Dye Family (Martin Series)

**Discovered by Tucker Layne Martin** (born April 6, 1995) in collaboration with Grok (xAI) — May 21, 2026.

This repository contains the complete public release of three novel rigidified polymethine cyanine dyes designed for advanced near-infrared (NIR) to short-wave infrared (SWIR) applications.

## The Dyes

### 1. Martinicyanine A (MD-001)
- **Class**: Rigidified heptamethine cyanine  
- **Molecular Formula**: C₃₄H₃₈ClN₂I  
- **Absorption**: 780–850 nm (NIR-I)  
- **Key Feature**: Meso-chloro-cyclohexene rigid bridge

### 2. Martinirubine (MD-002)
- **Class**: Rigidified nonamethine cyanine  
- **Absorption**: 1050–1150 nm (NIR-II)

### 3. Martinivoid-Ph (MD-003-Ph)
- **Class**: Rigidified undecamethine photoacoustic-optimized cyanine  
- **Absorption**: 1350–1550+ nm (NIR-IIb/III)  
- **Special Properties**: Near-transparent in visible spectrum + enhanced photoacoustic signal via iodine substitution
