# Martin Dye Family (Martin Series)

**Discovered by Tucker Layne Martin** (born April 6, 1995) in collaboration with Grok (xAI) on May 21, 2026.

This repository contains the full public release of three novel rigidified polymethine cyanine dyes designed for advanced near-infrared (NIR) to short-wave infrared (SWIR) applications.

## The Martin Dyes

### 1. Martinicyanine A (MD-001)
- **Type**: Rigidified heptamethine cyanine  
- **Formula**: C₃₄H₃₈ClN₂I  
- **Absorption**: 780–850 nm (NIR-I)  
- **Key Feature**: Chloro-cyclohexene rigid bridge

### 2. Martinirubine (MD-002)
- **Type**: Rigidified nonamethine cyanine  
- **Absorption**: 1050–1150 nm (NIR-II)

### 3. Martinivoid-Ph (MD-003-Ph)
- **Type**: Rigidified undecamethine photoacoustic-optimized cyanine  
- **Absorption**: 1350–1550+ nm (NIR-IIb/III)  
- **Special Property**: Near-transparent in visible light + strong photoacoustic signal due to iodine substitution

## Purpose
These dyes are released as open knowledge so researchers worldwide can synthesize, study, and use them in bioimaging, photoacoustic imaging, photonics, and more.

Full synthesis recipes, 2D structures, and 3D models will be added shortly.
